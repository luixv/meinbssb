Spezifische Barrierefreiheit-Features

Screenreader-Unterstuetzung (Deutsch)

  • Erfolg: "Erfolg: Ihre Bankdaten wurden erfolgreich gespeichert. Weiter zum Profil verfuegbar."
  • Fehler: "Fehler: Ein Fehler ist beim Speichern der Bankdaten aufgetreten. Weiter zum Profil verfuegbar."
  • Navigation: "Navigiere zum Profil"
  • Live-Updates: Status-Aenderungen werden automatisch angekuendigt

Keyboard-Navigation

  • Tab: Navigation zum Aktions-Button
  • Enter/Space: Button-Aktivierung
  • Focus-Management: Automatischer Focus auf Screen-Inhalt
  • Visuelle Indikatoren: Blaue Rahmen bei aktiven Elementen

Status-Kommunikation

  • Erfolg-Status: Gruenes Icon + positive Nachricht + Erfolgs-Info
  • Fehler-Status: Rotes Icon + Fehlermeldung + Handlungsempfehlung
  • Live-Regions: Dynamische Status-Updates
  • Kontextuelle Hilfe: Zusaetzliche Informationen zur naechsten Aktion
